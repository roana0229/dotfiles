## Setup on macOS

1. `xcode-select --install`
2. `sh -c "$(curl -fsSL https://raw.githubusercontent.com/roana0229/dotfiles/master/setup_macos.sh)";`
3. `sh -c "$(curl -fsSL https://raw.githubusercontent.com/roana0229/dotfiles/master/vim/setup.sh)";`

## Setup only vim

1. `curl -fsSL https://raw.githubusercontent.com/roana0229/dotfiles/master/vim/.vimrc_minimum > ~/.vimrc`
